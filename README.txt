# Coffee Tracker

A tiny iPhone Home Screen app for tracking coffee-machine payments.

Default settings:
- Advance paid: ₩10,000
- Cost per coffee: ₩500
- Total coffees: 20

How to use on iPhone:
1. Put these files somewhere you can open `index.html`, or host them on GitHub Pages / Netlify.
2. Open the page in Safari.
3. Tap Share.
4. Tap “Add to Home Screen”.
5. Open it like a normal app.

Your data is stored locally on the phone using localStorage.
